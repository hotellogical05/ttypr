# ttypr

**t**erminal **ty**ping **pr**actice

Small application to practice ASCII (for now) typing

Based on [Ratatui][Repo]

# Installation

```shell
cargo install ttypr
```

# Usage

Run the application - start typing.

ESC - to exit
